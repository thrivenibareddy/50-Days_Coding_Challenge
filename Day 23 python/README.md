# 🚖 Ride Complaint String Analysis

## 📌 Project Overview

This mini project focuses on analyzing ride complaint data using **pure string operations in Python** (without using advanced libraries like pandas). The goal is to clean, process, and extract useful insights from a single raw string containing multiple ride records.

---

## 🎯 Objectives

The project performs the following tasks:

1. Convert the entire string to lowercase
2. Count total number of complaints (records)
3. Count occurrences of **"late arrival"** issues
4. Count complaints from **Bangalore**
5. Extract the first driver's name
6. Standardize issue names (e.g., replace spaces with underscores)

---

## 🛠️ Technologies Used

* Python 🐍
* String Methods (split, count, replace, lower)

---

## 📊 Results

| Metric               | Value  |
| -------------------- | ------ |
| Total Complaints     | 3      |
| Late Arrival Issues  | 2      |
| Bangalore Complaints | 2      |
| First Driver Name    | ramesh |

---

## 📈 Key Insights

* Most complaints are related to **late arrivals**
* Bangalore has a higher number of complaints compared to other cities
* Data standardization improves consistency for further analysis
