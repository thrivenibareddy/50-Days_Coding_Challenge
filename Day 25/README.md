#  Day 25 — Strings & Tuples Coding Challenge (Python)

This project focuses on mastering **Strings and Tuples** using real-world student data scenarios.

---

##  Overview

In this challenge, multiple problems were solved to strengthen understanding of:

* String manipulation techniques
* Tuple operations and immutability
* Data cleaning and formatting
* Pythonic looping using comprehensions
* Efficient data extraction methods

---

## Problems Implemented

###  1. Student Name Formatting

Converted inconsistent student names into **proper case formatting**.

###  2. Count Total & Unique Subjects

Calculated:

* Total number of subjects
* Unique subjects using Python sets

###  3. Feedback Cleaning

Removed unnecessary leading and trailing spaces from feedback strings.

###  4. Combine Student Name & Course

Merged two tuples (`names` and `courses`) into a structured output:

```
('Anu-Python', 'Ravi-SQL', 'John-Power BI')
```

###  5. Extract Email Domains

Extracted domain names from student email addresses.

---

##  Technologies Used

* Python 3
* Strings
* Tuples
* Sets
* List/Tuple Comprehensions

---

##  Key Concepts Learned

* Immutable data handling using tuples
* String methods: `capitalize()`, `strip()`, `split()`
* Using `zip()` for parallel iteration
* Clean and readable Python coding practices
