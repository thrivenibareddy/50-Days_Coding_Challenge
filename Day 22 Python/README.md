# 📊 Sales Data Analysis (Python)

## 📌 Project Overview
This project demonstrates the use of Python variables and data types to process and analyze sales data for a single e-commerce order. It includes calculations for total price, discount, GST, and final bill generation.

## 🎯 Objectives
- Store data using Python variables  
- Identify and display data types  
- Perform price calculations  
- Apply discount logic  
- Add GST (18%)  
- Format and display final bill output  


## 🧾 Given Data

order_id = "ORD12345"
customer_name = "Rahul"
product_price = 799.99
quantity = 3
discount = 0.10
is_member = True

## 💡 Key Concepts Used
- Variables and Data Types (`str`, `int`, `float`, `bool`)
- Arithmetic Operations
- Conditional Expressions (`if-else`)
- String Formatting (`f-strings`)
- `type()` function

## 🏁 Conclusion
This project provides a beginner-friendly understanding of how Python can be used for basic data processing and financial calculations in real-world scenarios.
