**Overview**
This project focuses on transforming raw loan data into actionable insights through an interactive Excel dashboard.

It enables users to quickly evaluate portfolio health, identify high-risk segments, and track lending trends over time. By combining data cleaning, KPI creation, and visualization, the dashboard simulates real-world financial analysis used in banking and lending institutions.

**Data Processing**

Cleaned and standardized dataset

Created calculated fields:

Months on Book

Profit / Loss

Loan Performance Categories

 **Key Insights**

 Default risk increases with lower loan grades

 Debt consolidation is the most common loan purpose

 High DTI correlates with higher risk

 Certain states show higher loan concentration
