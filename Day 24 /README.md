# 🚕 Ride Booking Analysis (Tuple-Based Python Project)

## 📌 Project Overview

This project simulates a real-world **ride-booking data analysis** scenario (similar to Ola/Uber). The dataset is stored as Python **tuples**, and analysis is performed using basic Python concepts.

---

### 📊 Sample Data

```python
rides = (
    (101, "Ravi", "Bangalore", 250, 4.5),
    (102, "Anjali", "Hyderabad", 180, 4.2),
    (103, "Kiran", "Bangalore", 300, 4.8),
    (104, "Meena", "Chennai", 220, 4.0),
    (105, "Arjun", "Bangalore", 150, 3.9)
)
```

---

## 🎯 Objectives

* Count total number of rides
* Extract all rides from **Bangalore**
* Calculate total revenue from Bangalore
* Find the **highest-rated driver**
* Calculate **average rating per city**

---

## 🛠️ Technologies Used

* Python 🐍
* Tuples
* Loops & Conditional Statements
* Dictionary for aggregation

---

## 📊 Results

| Metric               | Value       |
| -------------------- | ----------- |
| Total Rides          | 5           |
| Bangalore Revenue    | 700         |
| Highest Rated Driver | Kiran (4.8) |

### ⭐ Average Ratings Per City

```
{
  "Bangalore": 4.4,
  "Hyderabad": 4.2,
  "Chennai": 4.0
}
```

## 📈 Key Insights

* Bangalore generates the highest revenue
* Kiran is the top-performing driver
* Ratings are consistent across cities
